# Product Guidelines: LeMon (Tipfy)

## Voice and Tone
- **Cyber-Techno:** The language should feel high-tech, futuristic, and "plugged-in." It should resonate with the crypto-native and gaming communities without being overly cold.

## UX/UI Principles
- **Aesthetic Overdrive:** Prioritize an immersive atmosphere. Visual flair, including animations, neon glows, and glitch transitions, should be used to reinforce the cyberpunk identity.
- **Dark Mode Mandatory:** The interface is built for the night. No light mode should be implemented to ensure neon accents (Cyan/Pink) remain high-contrast and impactful.
- **Real-time Responsiveness:** Every action deserves an immediate reaction. Donation alerts and UI feedback must be processed with ultra-low latency (<500ms) to maintain the "live" feel.
- **On-chain Transparency:** While we use user-friendly analogies for general concepts, every on-chain interaction (gas fees, transaction signing) must be explicitly communicated to ensure user trust.

## Content Style
- **Bridge the Gap:** Use user-friendly analogies (e.g., "Digital Receipt") for broad concepts while keeping the technical power accessible. Avoid overwhelming non-technical streamers with raw jargon unless necessary.
