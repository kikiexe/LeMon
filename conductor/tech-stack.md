# Technology Stack: LeMon (Tipfy)

## Languages
- **TypeScript:** Primary language for web application development (Frontend and API).
- **Solidity:** Language for writing smart contracts deployed on the Monad network.

## Frameworks & Libraries
- **React:** Component-based UI library.
- **TanStack Start:** Full-stack framework for routing, SSR, and API routes.
- **Vite:** Build tool and development server.
- **Tailwind CSS:** Utility-first CSS framework for rapid styling.
- **Framer Motion:** Animation library for cyberpunk effects and smooth transitions.

## Blockchain & Web3
- **Monad:** The target high-performance EVM-compatible Layer 1 network.
- **Foundry:** Development toolset for writing, testing, and deploying smart contracts.
- **Wagmi & Viem:** Low-level and high-level libraries for Ethereum/Monad interactions.
- **ConnectKit/AppKit:** Wallet connection management.

## Real-time & Backend
- **Ably:** Real-time messaging service for broadcasting donation alerts and overlay updates.
- **Nitro:** Underlying server engine for TanStack Start API routes.

## Database & Persistence
- **Neon:** Serverless PostgreSQL database.
- **Drizzle ORM:** TypeScript ORM for type-safe database interactions.

## Storage & Assets
- **Uploadthing:** Simple file uploading for creator assets.
- **Google Cloud Storage:** Long-term storage for platform data.
